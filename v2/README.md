# AI 協作流程控制器 v2（WIP / 開發中）

這是 v2 重構中的開發版（WIP）。目前核心目標是把「多窗口 AI 協作」流程固定化，並提供一鍵產生 CLI 指令與開場提示詞，讓你能快速在不同 AI CLI 工具間切換。

## 這版在做什麼（依照程式現況）
- 三窗口工作流程：架構師 A / 審查員 B / 執行者 C（另有秘書 D 輔助引導）
- 一鍵產生 CLI 指令 → 複製到剪貼簿 → 貼到 AI CLI 工具
- 多專案管理（專案下會自動建立資料夾結構與核心文件）
- 介面：三個 Tab
  1) 初始設定：建立專案結構與核心文件
  2) 啟動窗口：選 CLI 指令 + 產生 A/B/C/D 開場指令
  3) 工作流程：Step 1～6 引導你按流程貼給 A/B/C

## 支援哪些 CLI？
內建提供 Copilot / Claude / Codex 常用指令清單可選，也可用「自訂指令」模式手動輸入。
（注意：你要在自己的電腦先安裝好對應 CLI 才能用）

## 怎麼跑（開發版）
1) 安裝依賴（在 repo 根目錄）：
   pip install -r requirements.txt

2) 執行：
   python v2/AI討論工具_v2_WIP.py

## 專案建立後的資料夾結構（程式會自動建）
- _共用文件
- _窗口A_規劃
- _窗口B_審查
- _窗口C_執行
- _共識

同時會生成核心文件（例如 CLAUDE.md、PROJECT_STATE.md、CONSENSUS.md 等）。

## 重要注意（依照目前程式寫法）
- 設定檔會存到使用者 Desktop：AI流程控制器_config.json
- 開場提示詞內「共用文件」目前使用固定相對路徑：../_共用文件
  如果你的實際路徑不同，或你希望更彈性，請自行調整程式或設定策略。
